# 超星学习通翻译 v6 — 功能设计

---

## 1. 翻译模式

翻译默认为**替换模式**（英文替换中文），双语对照是独立功能。

```
翻译前:                    翻译后（替换）:             翻译后（对照）:
                           A. Waiting time             A. Waiting time
A. 等待时间                 B. Execution time           A. 等待时间
B. 执行时间                 C. Turnaround time          B. Execution time
C. 周转时间                 D. Average turnaround time  B. 执行时间
D. 平均周转时间                                        C. Turnaround time
                                                       C. 周转时间
```

| 状态 | 如何进入 | 显示效果 |
|------|---------|---------|
| 原文 | 初始状态 / 点"还原" | 原始中文 |
| 替换 | 点"翻译" | 纯英文 |
| 对照 | 点"对照" | 英文上 + 中文下（灰色小字）|

- "翻译"和"对照"是两个独立按钮
- 可以只翻译不看对照，也可以翻译后打开对照
- 点"还原"全部回到中文

---

## 2. 翻译记忆（持久缓存）

同一个中文句子翻译一次后，保存在本地。下次任何页面遇到相同句子，直接用缓存，不再调用 API。

- 缓存在 `chrome.storage.local`，跨页面、跨会话持久保留
- 翻译时显示缓存命中情况："翻译 35 项 · 28 缓存 · 7 新翻"
- 缓存可手动清空（设置面板中）

---

## 3. 单词本

翻译后，用户可以收藏不认识的英文单词，积累专业词汇。

### 3.1 收藏单词

翻译后，**Ctrl+Click** 任意英文单词 → 存入单词本。

流程：
```
鼠标移动到英文字上，按住 Ctrl 点击
    ↓
调用 DeepSeek 获取该词的中文释义
    ↓
存入单词本（word, meaning, context句子, 日期, 课程名）
    ↓
toast: "已收藏: scheduling → n. 调度"
```

### 3.2 单词本面板

点标签下方的"单词本"按钮 → 弹出面板：

```
┌────────────────────────────────┐
│  我的单词本              [清空] │
│                                │
│  ┌─ scheduling ──────────────┐ │
│  │  n. 调度                   │ │
│  │  原文: The reason for ...  │ │
│  │  课程: 操作系统原理          │ │
│  │  06-30                     │ │
│  │                       [×]  │ │
│  └────────────────────────────┘ │
│                                │
│  ┌─ exclusive ───────────────┐ │
│  │  adj. 独占的               │ │
│  │  原文: ...exclusive device │ │
│  │  课程: 操作系统原理          │ │
│  │  06-30                     │ │
│  │                       [×]  │ │
│  └────────────────────────────┘ │
│                                │
│  [导出 CSV]    [关闭]           │
│  已收藏 12 个单词               │
└────────────────────────────────┘
```

- 每张卡片：单词、释义、原文句子、课程名、日期
- 点 [×] 删除单个单词
- 点 [清空] 删除全部
- 点 [导出 CSV] 下载为 CSV 文件

### 3.3 课程识别

从页面标题中提取课程名：
- 页面标题: "操作系统原理（S）2026春季学期"
- 提取课程名: "操作系统原理（S）"
- 存入单词记录

## 5. 交互总览

| 操作 | 结果 |
|------|------|
| 点击"翻译" | 页面中文 → 英文（替换模式） |
| 点击"还原" | 页面恢复原始中文 |
| 点击"对照" | 切换双语对照 ON/OFF |
| 点击"单词本" | 打开单词本面板 |
| Ctrl+Click 英文单词 | 收藏该单词到单词本 |
| 点击 ⚙ | 打开设置面板（配 Key / 清缓存） |
| `Ctrl+Shift+T` | 翻译 / 还原 |
| `Ctrl+Shift+B` | 对照 / 替换 |
